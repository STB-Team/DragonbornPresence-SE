Skyrim True Believer runtime

StbRuntimeAdapter.h/.cpp владеет menu/combat sinks, SKSE task interface и std::jthread. Он переводит события в вызовы coordinator и гарантирует не более одной pending main-thread task.

Exception boundaries находятся внутри каждого event callback и scheduled task. Background thread не читает RE forms и не вызывает Discord SDK.

Discord runtime

DiscordSdkLoader отвечает за Windows bootstrap. DiscordPresenceClient реализует transport port, строит Activity, подавляет дубликаты и прекращает работу после неизвестного результата SDK.
