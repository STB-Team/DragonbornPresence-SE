Atomics используются только для значений, пересекающих thread/event boundary. Loading и gameplay state меняются на Skyrim main thread.

Граница с runtime

StbRuntimeAdapter вызывает только engine-independent public methods coordinator. Он не форматирует Presence. Coordinator не регистрирует RE event sinks и не планирует SKSE tasks.
