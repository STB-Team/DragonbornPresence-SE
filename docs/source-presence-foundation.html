Core-модели и JSON translation

Core задаёт независимый язык приложения. Config adapter переводит внешний JSON в эти модели и не отдаёт parser types наружу.

Модели

ФайлСодержимое
Config.hDefaults, Application ID, schema constant и location rules.
PlayerSnapshot.hOwned gameplay state.
LocationContext.hWorldspace, cell, parent locations и display name.
PresencePayload.hTransport-neutral Presence fields.
RefreshReason.hПричины refresh и log labels.

Чистые реализации

  • LocationAssetResolver.cpp — first-match ordered selection;
  • Difficulty.cpp — raw STB difficulty в подпись;
  • TextUtils.cpp — UTF-8 и ASCII-insensitive matching.
