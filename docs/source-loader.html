adapters/discord

Discord loader и Presence client

Два компонента разделяют Windows/DLL bootstrap и управление состоянием Discord Game SDK.

DiscordSdkLoader.h/.cpp

Публичные функции IsDiscordRunning и LoadDiscordSdk возвращают успех и optional diagnostic string. Реализация содержит registry read, process snapshot, путь DLL рядом с plugin, LoadLibraryExW и delay-load hook.

Открыть реализацию

DiscordPresenceClient.h/.cpp

Реализует IPresenceClient и владеет:

  • std::unique_ptr<discord::Core>;
  • transport health;
  • одним session timestamp;
  • последней успешной signature;
  • одной pending signature и callback timeout.

Основные методы: Initialize, RunCallbacks, UpdateActivity, Shutdown.
