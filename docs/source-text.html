UTF-8 и CP1251

Общая работа с UTF-8 живёт в core. Преобразование legacy-строк сборки STB остаётся во внешнем adapter.

include/DragonbornPresence/core/TextUtils.h
src/core/TextUtils.cpp

  • IsValidUtf8 — полная структурная проверка UTF-8;
  • LimitUtf8Bytes — ограничение поля Discord на границе code point;
  • ContainsAsciiInsensitive — поиск ASCII selector без аллокации lowercase-копии.
