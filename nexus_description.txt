[font=Tahoma][size=5][b]DRAGONBORN PRESENCE 3.1.4[/b][/size]
[size=3]DISCORD RICH PRESENCE ДЛЯ ТЕКУЩЕЙ СБОРКИ STB[/size][/font][/center]

[font=Tahoma]Внутренняя интеграция Skyrim True Believer. Плагин рассчитан на текущий [font=Courier New]STB.esp[/font], его MCM data-storage quest и Discord Assets команды STB. Сторонние сборки не поддерживаются.[/font]

[font=Tahoma][b]Автор оригинала:[/b] Fozar · [b]Адаптация для STB:[/b] Frem[/font]

[font=Tahoma][size=4][b]— Presence-контракт —[/b][/size][/font]

[font=Tahoma][list]
[*][b]details[/b] — выбранная сложность STB, например [i]🟢Приключение[/i].
[*][b]state[/b] — [i]lvl-14 💀-3 🎭-Атронах[/i].
[*][b]Большая картинка[/b] — первое совпавшее правило текущей локации.
[*][b]Загрузка[/b] — маленький asset [font=Courier New]loading[/font].
[*][b]Бой[/b] — asset [font=Courier New]combat[/font] и подсказка [i]В бою с <цель> (ур. <уровень>)[/i].
[*][b]Таймер[/b] — время текущего запуска Skyrim; загрузка и смена локации его не сбрасывают.
[/list][/font]

[font=Tahoma][size=4][b]— Источники STB —[/b][/size][/font]

[font=Tahoma][list]
[*]Смерти: Global [font=Courier New]aaMZgv_NowDeath[/font].
[*]Сложность: [font=Courier New]aaMZ_SelectedLevel_OfDifficulty[/font] в [font=Courier New]aaMZ_MCMDataStorage[/font].
[*]Камень: активное заклинание одного из 19 описаний камней STB.
[*]Локация: worldspace, location и parent cell игрока.
[/list][/font]

[font=Tahoma][size=4][b]— Runtime-файлы —[/b][/size][/font]

[font=Tahoma][font=Courier New]SKSE\Plugins\DragonbornPresence.dll
SKSE\Plugins\discord_game_sdk.dll
SKSE\Plugins\DragonbornPresence.json[/font]

Плагин не добавляет собственный .esp или Papyrus-скрипты. Рабочий JSON при пересборке DLL не перезаписывается.[/font]

[font=Tahoma][size=4][b]— Требования текущей сборки —[/b][/size][/font]

[font=Tahoma][list]
[*]Текущий Skyrim runtime сборки STB
[*]Соответствующие SKSE64 и Address Library
[*]Текущий [font=Courier New]STB.esp[/font]
[*]Установленный и запущенный Discord Desktop
[*]Discord Application ID [font=Courier New]1527543892151373937[/font] и assets команды STB
[/list][/font]

[font=Tahoma]Перед инициализацией проверяются обработчик [font=Courier New]discord://[/font], исполняемый файл и запущенный процесс Discord Desktop. Если клиент не установлен или не запущен, плагин отказывается от инициализации, [font=Courier New]discord_game_sdk.dll[/font] не загружается и Discord автоматически не запускается. Завершение клиента или ошибка SDK немедленно останавливают дальнейшую работу Presence.[/font]

[font=Tahoma][size=4][b]— Диагностика —[/b][/size][/font]

[font=Tahoma]Лог: [font=Courier New]%USERPROFILE%\Documents\My Games\Skyrim Special Edition\SKSE\DragonbornPresence.log[/font]

Для таймера все строки [font=Courier New]Presence updated; session_start=...[/font] в рамках одного запуска должны содержать одинаковое значение.[/font]
